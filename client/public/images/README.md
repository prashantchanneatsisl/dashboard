# Vessel Images

Place your vessel images in this folder. The dashboard will automatically load images from this directory.

## Supported formats
- JPG
- JPEG
- PNG
- GIF
- WebP

## Naming convention
Name your image files descriptively, e.g.:
- container-vessel-1.jpg
- oil-tanker-mumbai.jpg
- cargo-ship-ocean-pride.png

The system will automatically detect and display all images in this folder.
